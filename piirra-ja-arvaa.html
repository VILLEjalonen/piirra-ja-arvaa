🔥 Firebase-asetukset puuttuvat

Ennen kuin peli toimii, sinun täytyy:

1. Mene console.firebase.google.com → Luo projekti → Realtime Database → Luo tietokanta (test mode).

2. Kopioi projektisi firebaseConfig (Project Settings → Your apps → Web app).

3. Liitä arvot tiedoston alkuun muuttujiin:

const FIREBASE_CONFIG = { apiKey: "AIza...", authDomain: "projekti.firebaseapp.com", databaseURL: "https://projekti-default-rtdb.firebaseio.com", projectId: "projekti-id", storageBucket: "projekti.appspot.com", messagingSenderId: "123456789", appId: "1:123...:web:abc..." };

4. Julkaise tiedosto Netlifyssä tai GitHub Pagesissa.
